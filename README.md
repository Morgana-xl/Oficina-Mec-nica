Projeto Lógico de Oficina Mecânica

Estrutura do Banco de Dados:
- **Cliente**: Armazena dados dos clientes (id, nome, telefone)
- **Veículo**: Relaciona os veículos aos clientes
- **Serviço**: Lista os serviços prestados
- **Ordem_Serviço**: Registra cada ordem e data de execução
- **Item_Serviço**: Associa serviços às ordens, com quantidade e valores
